Steps for setting up the web interface for coronavirus project:

1. Copy all the files in the project folder

2. Open the connection.php and change the connection variables like server name, database name, username and password.

3. We can just copy the covid-19.sql ( after changing the database name) into the MariaDB. This will copy the required tables.

5. After this to enter into the main page, go to the front.php in the parent folder.

6. Setup is done, now can go through the website.
